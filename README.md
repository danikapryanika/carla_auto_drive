# carla_auto_drive
> Разработка моделей глубокого обучения для задачи автономной навигации

---

## 📖 О проекте

**CILDriveNet** — это мультимодальная нейронная сеть, разработанная для задачи автономной навигации транспортного средства в городской среде.

Модель принимает данные сразу из нескольких источников:

- 📷 RGB-камера;
- 📡 LiDAR (в формате Bird's-Eye View);
- 🚗 текущая скорость автомобиля;
- 🧭 навигационная команда (Follow, Left, Right, Straight).

На основе этих данных сеть предсказывает управляющие воздействия:

- угол поворота рулевого колеса (Steering);
- положение педали газа (Throttle).

Проект разработан в рамках выпускной квалификационной работы и реализован полностью на Python с использованием PyTorch.

---

# ✨ Основные возможности

- мультимодальная архитектура (RGB + LiDAR + Speed);
- Conditional Imitation Learning (CIL);
- использование предварительно обученной ResNet-18;
- рекуррентный модуль GRU для учета истории движения;
- автоматический сбор обучающих данных в CARLA;
- обучение с использованием Branch-Gated CIL Loss;
- дополнительная голова предсказания скорости (Speed Head);
- инференс в режиме реального времени;
- обнаружение препятствий при помощи LiDAR и встроенных сенсоров CARLA.

# 🧠 Архитектура модели

Модель состоит из нескольких независимых модулей.

```
RGB Camera
        │
        ▼
   ResNet-18 Encoder
        │
        │
LiDAR ─► CNN Encoder
        │
        │
Speed ─► Speed Encoder
        │
        ▼
     Feature Fusion
        │
        ▼
      GRU Module
        │
        ▼
 ┌─────────────────────┐
 │ Follow Lane Branch  │
 │ Turn Left Branch    │
 │ Turn Right Branch   │
 │ Go Straight Branch  │
 └─────────────────────┘
        │
        ▼
 Steering + Throttle
```

При обучении использовались:
